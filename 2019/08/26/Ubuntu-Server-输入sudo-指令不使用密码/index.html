Ubuntu Server 输入sudo 指令不使用密码

2019-08-26

sudo 指令在使用过程中需要反复输入密码，显得很麻烦，自然要取消这恼人的设定啊，哈哈哈

sudo cp /etc/sudoers .      # 备份默认配置到当前文件夹
sudo -i                     # 切换到root用户
visudo                      # 编辑sudo 配置文件，也可以用vi /etc/sudoers,不过使用vim编辑会提示不能保存只读文件

接下来就时修改配置，找到

root　　ALL=(ALL:ALL) ALL

在下边添加类似的一行,Your_Name是你要免密登陆的用户名

Your_Name = ALL=(ALL:ALL) NOPASSWD: ALL

接着将

%sudo ALL=(ALL:ALL) ALL

修改为

%sudo ALL=(ALL:ALL) NOPASSWD: ALL

保存后退出重新登陆，使用 sudo ls测试就会发现不需要输入密码了。
